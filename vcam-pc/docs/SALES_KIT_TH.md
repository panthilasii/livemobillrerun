## 📊 Metrics ที่ควร track

จดในไฟล์ Excel/Google Sheet เริ่มจากเดือนแรก

| Metric | ทำยังไง | ตั้งเป้าเดือนแรก |
|---|---|---|
| **คำสอบถามต่อวัน** | นับใน Line OA | ≥ 5 คน |
| **Conversion (สอบถาม → ซื้อ)** | คน-ซื้อ ÷ คน-สอบถาม | ≥ 20% |
| **Time to first sale** | จากตอบแชตจนรับเงิน | ≤ 2 ชม. |
| **Churn (ไม่ต่ออายุ)** | ลูกค้าหมดอายุ - ต่ออายุ | ≤ 30% |
| **Avg revenue / customer** | รายได้รวม ÷ จำนวนลูกค้า | ≥ 1,200 บาท |

ครบ 30 วัน revisit ตัวเลข → ปรับ template / ราคาให้ดีขึ้น

## 🚦 Pre-launch Checklist

ก่อนเปิดขายจริง เช็คให้ครบ

- [ ] Line OA `@npcreate` ตั้งค่า welcome message + auto-reply
- [ ] PromptPay QR code พร้อมใน Line OA bio
- [ ] โพสต์ขายอย่างน้อย 1 โพสต์ในกลุ่มลูกค้า
- [ ] วิดีโอสอนใช้ 5 นาที upload YouTube unlisted
- [ ] Customer bundle Windows + macOS upload Drive (link พร้อม)
- [ ] รัน `python tools/init_keys.py` 1 ครั้ง (ถ้ายังไม่ได้ทำ)
- [ ] ทดสอบออก license คนแรก (เครื่องตัวเอง = beta)
- [ ] Backup `.private_key` ไว้ที่ปลอดภัย (USB drive / encrypted cloud)
- [ ] เตรียมสลิปยืนยัน + ใบเสร็จ template (ถ้ามีลูกค้าขอ)

> **⚠️ สำคัญที่สุด** — ถ้า `.private_key` หาย → ออก key ใหม่ไม่ได้
> ถ้า `.private_key` หลุด → ลูกค้าโดนคนปลอม key ฟรี
> เก็บไว้ในที่ที่คุณเท่านั้นเข้าถึงได้
